Суть Проекта заключается в создании развлекательной игры Tetris при помощи языка программирования python,

написанного на языке программирования python с использованием библиотеки PyQT5

Запуск:1)Установить python версии 3.10+ (Тестировалось на версии 3.10.5)

2)Установить все библиотеки, перечисленные в requiremenets.txt файле с помощью(pip install -r requirements.txt)

3)Запустить приложение с помощью команды python main_weather.py

Описание модулей:
1)main_weather.py - основной код приложения, в нём реализована функция общения с ботом.

2)main.py - код в котором прописаны все кнопки

3)config.py - код в котором находятся все токены

Пояснительная часть:

Проект Telebot был создан Хакимовым Арсланом в рамках Лицея Академии Яндекса.

Проект был реализован на основе библиотек aiogram.
